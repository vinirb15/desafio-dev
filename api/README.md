## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Tests

```bash
# unit Tests
npm run test:unit

# e2e Tests
npm run test:e2e

# coverage
npm run test:coverage

```
